# Bóveda del Proyecto CEL (Consultora Económica Loja)

Esta bóveda es un entorno de trabajo para el desarrollo del proyecto de asignatura de Economía de la Innovación.

## Propósito
Realizar una valoración integral de la rentabilidad y probabilidad de éxito de una consultora económica en Loja, Ecuador.

## Contenido
- `index.qmd`: Tablero de control del proyecto.
- `01-lean-canvas.qmd`: Modelo de negocio inicial (basado en el borrador de Lean Canvas).
- `02-mercado.qmd`: (Pendiente) Análisis de demanda y competencia.
- `03-finanzas.qmd`: (Pendiente) Valoración financiera y flujos.

## Cómo trabajar
Toda la evidencia generada durante el ciclo debe almacenarse aquí en formato Quarto (`.qmd`) para asegurar la reproducibilidad y el cumplimiento de los estándares institucionales.
