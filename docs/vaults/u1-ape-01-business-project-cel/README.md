# Boveda del Proyecto CEL

Esta carpeta concentra el proyecto transversal Consultora Economica Loja.

## Puntos de Entrada

- `index.qmd`: navegacion y contexto del proyecto.
- `01-lean-canvas.qmd`: documento analitico principal.
- `assets/`: figuras, exportaciones y bibliografia de apoyo.
- `data/`, `logs/`, `scripts/`: trabajo operativo del proyecto.
- `templates/`: compatibilidad historica; la CSL canonica vive en `../../writing/apa.csl`.
